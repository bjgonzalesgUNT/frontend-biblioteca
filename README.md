# FRONTEND BASE (NEXTJS + NEXTUI + REDUX + AXIOS )

## PASOS

1. Copiar el archivo `.env.sample` y renombrarlo a `.env.local`
2. Instalar las dependencias

   ```bash
   npm install
   ```

## PRODUCCION

1. Generar el build de produccion

    ```bash
    npm run build
    ```

2. Levantar el build

    ```bash
    npm run start
    ```
